Tentang Love Calc

Love Calc adalah sebuah aplikasi sederhana yang membantu pengguna mengukur sejauh mana kedekatan mereka dengan pasangan yang mereka cintai.

Teknologi yang Digunakan

Aplikasi ini dibangun menggunakan teknologi berikut:

  • HTML, CSS, JavaScript
  • Bootstrap 5
  • RESTful API untuk mengambil data feedback
  • Local Storage untuk menyimpan data pengguna
Logika Algoritma

Love Calc menggunakan pendekatan sederhana untuk menghasilkan persentase cinta berdasarkan nama pengguna dan nama pasangan yang dimasukkan:

  • Menggabungkan kedua nama menjadi satu string
  • Menggunakan fungsi hash untuk mengubah string tersebut menjadi skor numerik
  • Skor numerik dibagi dengan modulo untuk mendapatkan persentase cinta antara 1-100%

Aplikasi ini dibuat untuk tujuan hiburan semata dan tidak dimaksudkan untuk digunakan sebagai penilaian yang serius terhadap hubungan cinta.
